# Single Cell ATAC-seq Analysis

This repository contains analysis performed for the Single Cell Bioinformatics course at Saarland University.

## Tools Used
- R
- ArchR
- chromVAR
- tidyverse

## Analysis Workflow
1. Data preprocessing and quality control
2. Fragment filtering and TSS enrichment filtering
3. Doublet detection
4. Peak calling
5. Dimensionality reduction using LSI
6. Clustering of cell populations
7. Transcription factor motif activity analysis
8. Integration with scRNA-seq data
